Before running pdollar.js, run the following command: 
    npm install         [This will install the dependencies required to run pdollar]
To run pdollar:
node pdollar.js 'arguments'

Usage: -> 
node pdollar.js -t <gesturefile>
e.g. node pdollar.js -t arrowhead.txt

node pdollar.js -r 

node pdollar.js -e <eventstream>
e.g. node pdollar.js -e exclamation_point_eventfile.txt

** Keep the gesture files in gestureFiles folder and event files in eventFiles folder 
in order to run the program
